# WIT E-Commerce

A simple e-commerce frontend project built with React and Vite.

Live demo: [https://wit-e-commerce.vercel.app](https://wit-e-commerce.vercel.app)

## Features

- Product listing and product detail pages
- Shopping cart flow
- Login and sign up screens
- Order summary and previous orders pages
- State management with Redux

## Technologies

- React
- Vite
- Redux
- React Router
- Tailwind CSS
- Axios

## Installation

```bash
npm install
npm run dev
```

Then open the app in your browser:

```bash
http://localhost:5173
```

## Other Commands

```bash
npm run build
npm run lint
npm run preview
```
